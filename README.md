# leetRoute: drive awesome

> A custom route planning app for recreational and spirited drivers

[![Version](https://img.shields.io/badge/version-0.1.7-blue)](https://github.com/sigilpunk/leetRoute)
[![Status](https://img.shields.io/badge/status-alpha-orange)](https://github.com/sigilpunk/leetRoute)

## what's a leetRoute?

leetRoute picks your route based on criteria **you** set, optimizing for the driving experience rather than just the fastest path. Configure your ideal drive with:

- **Number of turns** - more turns, more fun
- **Maximum turn radius** - find those tight corners (or not!)
- **Number of straightaways** - overtake that kia soul bro
- **Highest average speed limit** - vroom
- **...and more!**

## current status

leetRoute is still very much in the works. A basic demo is currently live and being updated over at [leetroute.vercel.app](https://leetroute.vercel.app), however **none of the promised core functionality exists... yet!**

This is a working preview of the UI and engine - the optimized route engine is coming soon.

## current features

### route preview
- Built-in map viewer to preview your route
- Google Maps URLs for familiar navigation

### export formats
Once calculated, export your custom route to:
- **GPX** (GPS Exchange) - Universal GPS format
- **KML** (Keyhole Markup Language) - For Google Earth and Maps
- **JSON** - Complete route data from the engine (for the nerds out there ðŸ¤“)

## stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Python Flask (with Jinja2 templating) (yea yea i know)
- **Deployment**: Vercel
- **Location Services**: Photon Geocoding API
